# UnitTestCooperativa - Pruebas .NET 8 con NUnit

Este proyecto demuestra el uso de NUnit para pruebas unitarias en un proyecto .NET 8. Est√° configurado para ser estable, moderno y evitar conflictos de paquetes comunes.

## Uso

Para trabajar con este proyecto, puedes usar los siguientes comandos de la CLI de .NET en tu terminal:

```bash
# Restaura las dependencias del proyecto
dotnet restore

# Compila el proyecto y sus dependencias
dotnet build

# Ejecuta las pruebas unitarias
dotnet test
```
